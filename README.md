# Simple-banking-system-c
A basic console-based banking system written in C that allows users to create an account, deposit money, withdraw money, and check their balance.
This is a beginner-friendly, console-based banking system developed in the C programming language.  
The program demonstrates basic C concepts such as functions, pointers, conditional statements, and loops.
